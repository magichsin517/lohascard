## 如果要從你的 Mac 直接改 code

目前沙盒的權限問題讓你 Mac 本機那份 `.git/` 可能是壞的。想從 Mac 直接改 code + push,做一次清理:

```bash
cd "/Users/yunghsin/Documents/文稿 - 王永興的iMac/working/2026/lohascard"
sudo rm -rf .git
git clone https://github.com/magichsin517/lohascard.git /tmp/lohas-fresh
mv /tmp/lohas-fresh/.git .
rm -rf /tmp/lohas-fresh
git status  # 看看有沒有你本機沒 commit 的改動
```

然後以後在這資料夾改檔 → `git add .` → `git commit -m "..."` → `git push`,網站 30-60 秒自動更新。
