## 回滾

**回程式碼:**
```bash
git log --oneline   # 找想回去的 commit
git revert <commit>
git push
```

**回資料(把解析出的單堂課全部刪掉):**
```sql
-- Supabase SQL Editor
delete from activities where tags @> array['單堂課'];
```

**把月課表 parent 恢復可重新解析(清掉 tag):**
```sql
update activities
set tags = array_remove(array_remove(array_remove(tags, '無課表附件'), '解析失敗'), '解析0堂')
where source_name = '教育部樂齡學習網';
```
